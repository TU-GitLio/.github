### 1. Clone Repository

```
https://github.com/2023-Winter-Bootcamp-TeamH/buyself-docker
```

### 2. Install Pacakages

```
$ cd buyself-frontend
$ npm run build
```

### 3. Env Settings

 - Add /buyself-docker/settings/.env file

```bash
### buyself-backedn/config
# === Database ===
DB_USERNAME =
DB_PASSWORD =
DB_HOST =
DB_SCHEMA =
DB_PORT =
# === S3Bucket ===
ACCESS_KEY_ID =
SECRET_ACCESS_KEY =
S3_BUCKET_REGION =

```

### 4. Run Docker

```
$ docker-compose -f docker-compose.prod.yml up --build   # build images and run containers
```
